# منصة الإشراف التربوي

## الملفات
- `index.html` الواجهة الرئيسية
- `assets/styles.css` التنسيقات
- `assets/app.js` المنطق الكامل وربط Supabase
- `manifest.webmanifest` ملف PWA
- `sw.js` Service Worker
- `offline.html` صفحة عدم الاتصال
- `supabase.sql` ملف قاعدة البيانات وRLS
